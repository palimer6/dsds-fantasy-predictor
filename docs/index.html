✨ DSDS Fantasy Predictor ✨

A tool to show what times would cause who to win in DSDS Fantasy

Features

  • Adjust the sliders on top to see who would win in any scenario you want.
  • Click the Reset button by a game slider to reset that game's time and have it not be included in score calculations.
  • Click the Set button at the end of any row to automatically set every remaining game's time to that person's exact guess. Use this to see if you can still win!
    • Click the Set button while holding shift to only set games that have not been set otherwise. (A game that has not been set will have "‒:‒‒:‒‒" listed as its time)
  • Click any row to highlight it. That way you can more easily track it while it changes positions.
  • Click the button in the top right to toggle between dark and light mode. When loading the page, your device's preference will automatically be applied.
  • If you find a combination of times that result in two or more people being tied, their rows will be highlighted together. Either in the placement colors if they are in the top 3, or in cyan.
  • Click the button in the top right to select which games' columns you want to show in the table.
    • When all games would make the table wider than your device, the game columns are hidden automatically.
  • Click the button, the escape key, or anywhere outside the info box to close it.
